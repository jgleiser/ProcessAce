Local / Custom Models (Ollama, LocalAI, etc.)

You can connect to a local or custom LLM by overriding the Base URL above. For instance, if you run Ollama locally, you can configure a dummy API key for OpenAI, select OpenAI as the provider, and set the Base URL to http://localhost:11434/v1.
